# @trainhive/shared

Shared utilities, types, and constants used across all TrainHive applications.

## Usage

```typescript
import { someUtil } from '@trainhive/shared/utils';
import { SomeType } from '@trainhive/shared/types';
import { CONSTANTS } from '@trainhive/shared/constants';
```

## Structure

- `src/types/` - Shared TypeScript type definitions
- `src/utils/` - Shared utility functions
- `src/constants/` - Shared constants
